# Pet Expression Storyboard

Pet uses four-frame loops for each eye expression:

- F1: base/readable pose
- F2: preparation or reaction
- F3: emotional peak
- F4: rebound or aftertaste

| Expression | F1 | F2 | F3 | F4 |
|---|---|---|---|---|
| Idle 平静 | 双眼大而圆，瞳孔居中 | 瞳孔看左 | 瞳孔看右 | 瞳孔轻微向下再回中 |
| Blink High 高位眨眼 | 正常睁眼 | 上眼睑快速下压 | 在较高位置闭成细线 | 重新睁开 |
| Blink Low 低位眨眼 | 正常睁眼 | 下眼睑/眼形整体下沉 | 在较低位置闭成细线 | 重新睁开 |
| Happy 开心 | 正常偏大的眼睛 | 上眼睑压下，眼睛变扁 | 变成弯弯的笑眼，像倒 U | 回到半笑眼 |
| Glee 雀跃 | 大眼睛，略上扬 | 双眼向上弹一下 | 眯成更大的弯弯笑眼，带闪光 | 回弹到开心大眼 |
| Curious 好奇 | 从平静正常睁眼开始，瞳孔居中 | 像注意到什么，眼睛变大并略微上抬 | 双眼圆睁，瞳孔更大，接近 😯 的惊奇感 | 仍睁大并看向一侧，保留好奇余韵 |
| Sleepy 犯困 | 半睁，眼皮重 | 上眼睑继续下垂 | 快闭上，只剩小缝 | 稍微撑开一点 |
| Sleeping 睡着 | 双眼闭成细线 | 细线轻微下沉 | 细线轻微上浮，Z 出现/变亮 | 回到平稳闭眼 |
| Hungry 期待 | 双眼略大，瞳孔向上/向用户 | 眼睛轻微放大 | 瞳孔靠近中间，像期待 | 眼睛下沿软下来 |
| Excited 兴奋 | 双眼大睁 | 双眼放大并上弹 | 高光/星星更亮，瞳孔上移 | 回弹但仍睁大 |
| Dizzy 晕乎 | 双眼内有螺旋 | 螺旋扩大/偏移 | 螺旋错位，眼形晃动 | 回到原螺旋但位置偏一点 |
| Low Battery 低电量 | 双眼低垂、暗红、很窄 | 眼睛继续下沉 | 眼睛短暂变暗，空电池符号闪 | 微微撑开但无力 |
| Charging 充电中 | 绿色双眼，瞳孔/闪电稳定 | 眼睛亮度升高 | 闪电变粗/更亮，眼睛微放大 | 亮度回落 |
| Proud 得意 | 双眼半眯，眼神偏上 | 眼角略上挑 | 一只眼更眯，像“哼” | 回到半眯得意 |
| Grumpy 不爽 | 双眼斜切，眉压低 | 眼睛更窄 | 瞳孔偏向侧边，像瞪人 | 保持压低但稍回弹 |
| Skeptic 怀疑 | 左右眼不对称，一大一小 | 小眼更窄，大眼轻微转向 | 双眼看向同一侧 | 回到不对称基准 |
| Worried 担心 | 双眼内收，眼皮向上拱 | 瞳孔轻微颤动 | 眼睛更圆、更靠中 | 回到担心半睁 |
| Love 喜欢 | 粉色爱心眼 | 爱心放大 | 爱心最大并上弹 | 爱心缩回但保持粉色 |
| Annoyed 烦躁 | 半眯，眼神斜向一边 | 一只眼更窄 | 双眼向旁边瞥 | 回到冷淡半眯 |
| Scared 害怕 | 双眼圆大，瞳孔靠内 | 眼睛更大 | 瞳孔颤动/偏下 | 回到大眼但仍紧张 |

Implementation note: keep F1 recognizable as a static icon and make F3 the clearest emotional keyframe.
